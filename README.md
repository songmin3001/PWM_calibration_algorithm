# PWM_calibration_algorithm

Arduino 기반 4륜 자율주행 미로 탐색 차량의 **모터 PWM 보정 + 초음파 장애물 회피** 코드입니다.

----

## 배경 및 문제 정의

초기 설계에서는 초음파 센서를 이용한 장애물 감지·회피 로직만 구현하면 충분할 것으로 예상했습니다.  
그러나 실제 테스트에서 **4개 바퀴의 속도 편차**로 인한 직진 주행 불안정 문제가 발견되었습니다.

| 편차 항목 | 측정 결과 |
|-----------|-----------|
| 좌/우 바퀴 속도 편차 | 약 **15%** |
| 전/후 바퀴 속도 편차 | 약 **10%** |
| 초기 완주 시간 | **2분 23초** (목표 대비 2배 이상) |

동일한 PWM 신호를 보내도 각 모터 개체 차이로 인해 실제 회전 속도가 달라지고,  
차량이 목표 경로를 벗어나거나 미로 벽면에 충돌하는 현상이 빈번했습니다.

**해결 방향:** 하드웨어 교체 없이, 각 바퀴의 속도 편차를 정량화해 개별 PWM 보정값을 적용하는 소프트웨어적 해법 채택.

---

## 동작 원리

```
실측 편차 (엔코더 100회 이상 측정)
        ↓
보정 계수 K_FL / K_FR / K_RL / K_RR 산출
        ↓
correctedPWM(BASE_SPEED, K_xx) → 각 모터 개별 PWM 출력
        ↓
직진 안정성 확보 + 초음파 장애물 감지 시 회피 실행
```

---

## 하드웨어 구성

| 부품 | 사양 |
|------|------|
| MCU | Arduino Mega 2560 (권장) |
| 모터 드라이버 | L298N × 2 |
| 구동부 | DC 모터 × 4 (4WD) |
| 센서 | HC-SR04 초음파 센서 |
| 측정 도구 | 광학식 엔코더 + RPM 측정기 |

### 핀 배선

| 핀 | 역할 |
|----|------|
| D5 (ENA) | 좌전 모터 PWM |
| D6 (ENB) | 우전 모터 PWM |
| D9 (ENC) | 좌후 모터 PWM |
| D10 (END) | 우후 모터 PWM |
| D22~D29 | 모터 방향 제어 (IN1~IN8) |
| D11 (TRIG) | 초음파 송신 |
| D12 (ECHO) | 초음파 수신 |

> 실제 배선에 따라 `#define` 핀 번호를 수정하세요.

## 주요 파라미터

```cpp
const int   BASE_SPEED = 180;      // 기준 속도 (0~255)
const float K_FL = 1.00;           // 좌전 보정 계수 (기준)
const float K_FR = 0.85;           // 우전 (좌/우 15% 편차 반영)
const float K_RL = 0.90;           // 좌후 (전/후 10% 편차 반영)
const float K_RR = 0.90 * 0.85;   // 우후 (좌우 + 전후 편차 누적)
```

| 파라미터 | 기본값 | 설명 |
|----------|--------|------|
| `BASE_SPEED` | 180 | 기준 PWM (실환경에 맞게 조정) |
| `K_FL` | 1.00 | 좌전 계수 — 기준값으로 고정 |
| `K_FR` | 0.85 | 우전 계수 — 좌우 편차 15% 반영 |
| `K_RL` | 0.90 | 좌후 계수 — 전후 편차 10% 반영 |
| `K_RR` | 0.765 | 우후 계수 — 두 편차 누적 반영 |
| `OBSTACLE_DIST_CM` | 20 | 장애물 감지 거리 (cm) |
| `turnRight(600)` | 600ms | 약 90° 회전 시간 |

---

## 캘리브레이션 방법

1. **속도 측정**  
   광학식 엔코더로 각 바퀴의 RPM을 100회 이상 측정하고 평균값 산출

2. **보정 계수 계산**  
   ```
   K_xx = 가장 느린 바퀴 RPM / 해당 바퀴 RPM
   ```
   > 가장 빠른 바퀴(여기선 좌전)를 기준(1.00)으로 삼고 나머지를 상대적으로 낮춤

3. **직진 테스트**  
   3~5m 직선 주행 후 경로 이탈 여부 확인 → 계수 미세 조정

4. **회전각 보정**  
   `turnRight()` / `turnLeft()` 의 ms 값을 조정해 정확한 90° 회전 확보

---

## 주요 함수

| 함수 | 설명 |
|------|------|
| `correctedPWM(base, k)` | 기준 속도 × 보정 계수 → 0~255 클램핑 |
| `driveForward(spd)` | 4륜 전진 (보정값 적용) |
| `driveStop()` | 전체 모터 정지 |
| `turnRight(ms)` | 제자리 우회전 (ms 동안) |
| `turnLeft(ms)` | 제자리 좌회전 (ms 동안) |
| `getDistanceCM()` | 초음파 거리 측정 (cm), 타임아웃 시 999 반환 |

---

## 장애물 회피 로직

```
거리 < 20cm 감지
    → 정지 (100ms)
    → 후진 (300ms)
    → 정지 (100ms)
    → 우회전 90° (600ms)
    → 전진 재개
```

미로의 좁은 통로 특성을 고려해 후진 후 회전하는 방식으로 벽면 충돌을 방지합니다.
